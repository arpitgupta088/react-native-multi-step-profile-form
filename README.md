# React Native Multi-Step Profile Form

## Overview
A mobile application built using React Native, TypeScript, and Redux Toolkit that allows users to create, edit, view, and delete user profiles using a 3-step form workflow. The app demonstrates centralized state management, clean navigation flow, and modular architecture without using a backend.

## Features
- 3-step multi-page form (Basic Info, Address, Summary)
- Redux Toolkit-based CRUD operations
- TypeScript interfaces for strong type safety
- Profile detail screen
- Edit and delete functionality
- Clean and modular folder structure

## Tech Stack
- React Native (Expo)
- TypeScript
- Redux Toolkit
- React Navigation
